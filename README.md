# LifeSim CLI

A simple command-line life simulation game written in Rust.  
You start at age 0 and make choices each year that affect your stats until the game ends.

---

## Features

- Start from age 0
- Year-by-year decision system
- Player stats:
  - Health
  - Happiness
  - Money
- Actions:
  - Study
  - Work
  - Party
  - Gym
  - Do nothing
- Random events (money gain, sickness)
- Game ends when:
  - Health reaches 0
  - Happiness reaches 0
  - Age reaches 100

---

## Requirements

- Rust (latest stable recommended)
- Cargo (comes with Rust)
